# Calculadora Simples

Calculadora básica desenvolvida em HTML, CSS e JavaScript, com suporte para operações matemáticas comuns.

## Funcionalidades

* Adição, subtração, multiplicação e divisão
* Digitação de números ilimitados
* Suporte a ponto decimal
* Botão `C` para limpar o display
* Botão `<-` para apagar o último dígito
* Botão `=` para calcular o resultado

## Estrutura do Projeto

```
index.html      
index.css       
calculator.js   
display.js
index.js
```

1. Abrir `index.html` em um navegador.
